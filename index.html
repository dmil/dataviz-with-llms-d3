Methodology

Articles were collected from BillTrack50 and converted to text embeddings using OpenAI's embeddings API endpoint. The distances between embeddings were then visualized in a two-dimensional t-sne plot using the scikit-learn package in Python. A two-dimenional hoverable scatter plot was then generated in D3 with the help of Open AI's ChatGPT 3.5 using the following prompt:

I have a CSV file named bills-with-nlp.csv with the following columns: State,Bill Number,title,Summary,Bill Progress,Last Action,publish_date,keywords,combined,ada_embedding,embedding,x,y. Show me the code to make a scatterplot in D3 with a tooltip. Each dot should be placed at x,y. When a user hovers on the tooltip, they should see the title and publish date. When they click on a dot, it should navigate to the url. And please add a search box that will let users filter by searching the title column.
